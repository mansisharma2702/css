CSS PSEUDO-CLASS..!

The CSS pseudo-classes allow you to style the dynamic states of an element uch as hover, active and focus state, as well as elements that are existing in the document tree but can't be targeted via the use of other seelctors without adding any IDs or classes to them, for example, tragetting the first or last child elements.

A pseudo-class starts wit h a colon (:). Its syntax can be given with:

selector:pseudo-class { property: value; }
