CSS Pseudo-Elements..!

The CSS pseudo-element allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS-3 introduced a new double-colon (::) syntax for pseudo-elements to distinguish between them and pseudo-classes.

Selector::pseudo-element { property: value; }
